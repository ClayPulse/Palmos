Vibe Dev Flow is a vibe coding service that generates, builds, and publishes
Pulse Apps using cloud-based coding agents with both user-friendly UI and backend API.

This API exposes a **streaming Server-Sent Events (SSE)** endpoint that:
- Creates a remote workspace
- Runs AI coding agents
- Builds and publishes a Pulse App
- Streams structured progress updates in real time

Clients should consume the response as an SSE stream and parse each
`data:` event as JSON.
