# Get Pulse Editor API Key

You will need an API key to use Pulse Editor workflow and automation via REST APIs. You can get your API key from your Pulse Editor account settings:

1. Sign up or log in to your Pulse Editor account at [https://pulse-editor.com](https://pulse-editor.com).
2. Go to [`developer`](https://pulse-editor.com/home/settings/developer) section under your account settings.
3. If you do not have beta access, submit an application form to request and join beta access here: [https://pulse-editor.com/beta](https://pulse-editor.com/beta)
4. Once you have beta access, you will be able to create and see your API key in the `developer` section. Create an API key, copy the API key and keep it secure.
   
You can now use this API key to login [Pulse Editor CLI](/guide/develop-extension-apps/cli) or make REST API calls to interact with Pulse Editor programmatically.
