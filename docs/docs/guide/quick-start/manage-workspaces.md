# Manage Workspaces

## What is a Workspace
A workspace is a computing environment either with VM or local computer. Not all apps require a computing environment. Check the app card for more info.

Local workspace is available on desktop builds. For web and mobile versions, please use a [Cloud Workspace](#cloud-workspace). If on desktop, you can switch between cloud managed workspace and local workspace.

## Local Workspace
Use your local PC as the main computing environment. 

## Cloud Workspace
You can start a Cloud Workspace after you have chosen a project.

You can choose from the following specs configurations. 

### Computing
 - 1 vCPU, 2GB RAM
 - 2 vCPU, 4GB RAM
 - 4 vCPU, 8GB RAM


### Storage
Up to 512 GB.

:::tip
> For **free plan** subscription, you can choose up to **1 workspace** with up to **1 vCPU, 2GB RAM**, and up to **24GB storage**.
:::
